# Bensz CV Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在仓库中新增 `packages/bensz-cv/` 与 `projects/cv-01/`，复刻只读源简历的中英文样式与构建体验，并提供像素级 PDF 比对和去隐私公开示例。

**Architecture:** 先把只读源简历拆成“公共样式与资源”沉淀到 `packages/bensz-cv/`，再用 `projects/cv-01/` 承载双语示例、头像与参考文献。统一构建入口参考 `bensz-thesis` 的 PDF/compare 能力与 `bensz-paper` 的项目级 wrapper 形态，确保完整仓库和单项目场景都能调用同一条官方链路。

**Tech Stack:** XeLaTeX、BibTeX、Python 3、Pillow、pdftoppm、VS Code LaTeX Workshop
