# nsfc-humanization

去除 NSFC 标书中的 AI 机器味，使文本读起来像资深领域专家亲笔撰写。

## 适用场景

- 用 AI 起草了标书段落，需要去除程式化表达
- 文本逻辑正确但读起来"太像 AI 写的"
- 希望在不改变内容的前提下提升文字质感

不适用于：非 NSFC 标书内容 / 需要修改格式 / 需要补充新内容。

## 快速开始

直接将需要润色的文本发给 AI，并说明使用本技能：

```
请用 nsfc-humanization 技能润色以下段落：

[粘贴你的标书文本]
```

支持纯文本和 LaTeX 混合文本，LaTeX 命令和结构不会被修改。

## 使用示例

### 示例 1：纯文本

输入：
```
本研究首先对现有方法进行了系统综述，其次分析了其局限性，最后提出了新的解决方案。该方案具有以下优点：第一，计算效率高；第二，适用范围广；第三，易于实现。因此，本研究具有重要的科学意义和应用价值。
```

输出：
```
现有方法的局限性已有文献充分讨论，但针对高维稀疏场景的系统性解决方案仍付阙如。本研究在此基础上提出新方案，其核心优势在于计算效率与适用范围的兼顾——这在以往工作中往往难以同时实现。实现路径的简洁性则进一步降低了推广门槛。
```

### 示例 2：LaTeX 混合文本

输入：
```latex
\subsection{研究意义}

本研究的意义主要体现在以下几个方面。首先，从理论层面来看，本研究填补了领域内的空白。其次，从应用层面来看，研究成果可以直接指导临床实践。最后，本研究为后续工作奠定了基础。
```

输出：
```latex
\subsection{研究意义}

该领域长期缺乏系统性的理论框架，本研究的工作在一定程度上填补了这一空白。更值得关注的是其转化潜力：所得结论可直接对接临床决策流程，而非停留在机制阐释层面。这也为后续干预研究提供了可操作的理论支点。
```

## 常见问题

**Q：会修改 LaTeX 命令吗？**
不会。所有 LaTeX 命令、环境、宏、注释、空行、缩进均保持原样。

**Q：会补充新内容吗？**
不会。本技能只润色文字表达，不新增任何实质性内容。

**Q：可以处理整篇标书吗？**
建议按段落或小节分批处理，便于逐段核查润色效果。

**Q：润色后需要人工审核吗？**
建议审核，尤其是专业术语的使用是否符合本领域习惯。
