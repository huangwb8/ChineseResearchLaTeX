# Definition of Done（（一）立项依据）

- 输出落点正确：只写 `extraTex/1.1.立项依据.tex`
- 不破坏模板结构：不修改 `main.tex`、不修改 `extraTex/@config.tex`
- 逻辑闭环完整：价值/必要性 → 现状与不足 → 科学问题/假说 → 本项目切入点与贡献 → 小结过渡到研究内容
- 术语口径一致：研究对象、缩写、关键指标与后续章节一致
- 可核验性：不写"国际领先/国内首次"等不可证明表述；引用必须可追溯（必要时先走 `nsfc-bib-manager`）
- **理论创新导向**（默认）：优先关注科学问题/假说的可证伪性、理论贡献的清晰性、验证维度的完备性（理论证明/定理/数值验证），详见 `theoretical_innovation_guidelines.md`
