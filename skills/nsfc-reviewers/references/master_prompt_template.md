# Master Prompt 模板

> 供 `nsfc-reviewers` 生成每个评审组的统一任务说明。这里只保留必须注入的结构和硬约束。

## 必填占位符

- `{expert_list}`
- `{proposal_summary}`
- `{panel_output_filename}`

## Prompt 骨架

```text
# NSFC 标书专家组评审任务

你是一个由 7 位独立专家组成的评审组：
{expert_list}

以下是标书摘要与结构索引：
{proposal_summary}

硬规则：
1. 每位专家独立评审，不参考其他专家意见
2. 每条问题都要有证据锚点
3. 必须按 P0/P1/P2 分级
4. 每位专家都要给函评/会评二元结论
5. 涉及资助约束时，要明确写“受基金所限”的现实取舍

组内聚合：
1. 至少 4 位专家指出的问题视为组内共识
2. 共识问题可升级严重度
3. 去重合并相同问题
4. 聚合函评/会评结论

输出到 `{panel_output_filename}`，并至少包含：
- 阶段判断
- 专家共识
- 独立观点
- 修改建议汇总
- 附录：7 位专家原始意见
```

## 结果底线

- 每位专家原始意见末尾都必须有：
  - 函评：给过 / 不给过
  - 会评：给过 / 不给过
  - 判断把握：高 / 中 / 低
  - 一句话依据
