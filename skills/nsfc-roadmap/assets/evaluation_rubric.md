# 技术路线图评估评分表（Rubric）

本评分表用于“独立评估”每轮渲染结果。评估时只基于当前图片与本表，不引入历史轮次结论。

## 维度与门槛

- **逻辑完整性（P0）**：是否覆盖研究内容的关键模块？是否有清晰输入与产出？是否存在孤立节点/断链？
- **结构层次（P0/P1）**：阶段/模块是否清晰分层？主流向是否一致（上→下/左→右）？
- **可读性（P0/P1）**：A4 打印是否无需放大？字号是否 ≥ 12pt 等效？文字是否溢出/重叠？
- **视觉规范（P1/P2）**：配色是否克制（主色 ≤3 + 辅助灰）？间距是否均匀？对齐是否整齐？
- **风险可见（P1）**：是否出现风险/对照/备选方案节点？是否能看出“想过失败怎么办”？

## 严重度口径

- **P0**：会被评审直接扣分（缺研究内容、逻辑断裂、不可读）
- **P1**：显著影响专业印象（布局混乱、刺眼配色、信息拥挤、字号偏小）
- **P2**：一般优化（间距/对齐、箭头/边框细节、色彩微调）
