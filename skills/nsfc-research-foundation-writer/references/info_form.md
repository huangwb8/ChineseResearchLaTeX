# NSFC 2026 写作信息表（面向（ 三）研究基础）

1. **前期基础（证据链素材）**：
   - 已发表/已接收论文（请给出题目、期刊/会议、年份、DOI/链接；若不便公开可给“可核验线索”）
   - 专利/软件著作权/数据集/平台/原型
   - 预实验/初步结果（现象+数据概况+可复现实验条件）
2. **团队与分工**：
   - 关键成员能力与负责模块
   - 合作单位与贡献（数据/样本/平台/算法/临床/工程验证）
3. **条件与资源**：
   - 平台、设备、算力、数据/样本来源、伦理/合规路径与周期
4. **风险清单（至少 3 条）**：
   - 技术风险（方法不收敛/模型泛化不足/实验失败等）
   - 进度风险（样本入组慢/审批周期长等）
   - 资源风险（设备故障/算力不足/合作不稳定等）
   - 每条风险给：早期信号 + 备选方案/降级路线
