# thesis-jlau-doctor 项目指令

本项目是吉林农业大学博士学位论文模板示例，属于 `bensz-thesis` 产品线。

- 公共样式优先维护在 `packages/bensz-thesis/styles/bthesis-style-thesis-jlau-doctor.tex`
- profile 优先维护在 `packages/bensz-thesis/profiles/bthesis-profile-thesis-jlau-doctor.def`
- 示例正文、封面字段和公开演示内容优先维护在本项目 `extraTex/`
- 构建优先使用 `python packages/bensz-thesis/scripts/thesis_project_tool.py build --project-dir projects/thesis-jlau-doctor`
- 不要把其它学校 thesis 模板的私有正文、私有图片或验收对比图复制到本项目
