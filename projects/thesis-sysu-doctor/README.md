# thesis-sysu-doctor

`thesis-sysu-doctor` 是 `bensz-thesis` 的中山大学博士论文公开示例项目。

说明：

- 版式源自已验证的 SYSU 博士论文样式
- 当前正文、图表和附录均为公开重写的演示内容
- 作者统一使用“冯宝宝”
- 项目仅用于模板演示，不包含任何真实论文正文或私有研究数据

构建方式：

```bash
python packages/bensz-thesis/scripts/thesis_project_tool.py build --project-dir projects/thesis-sysu-doctor
```

只打开项目子目录时，可执行：

```bash
python scripts/thesis_build.py
```
