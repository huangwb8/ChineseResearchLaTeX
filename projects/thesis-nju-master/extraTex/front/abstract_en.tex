\njuAbstractTitle{Nanjing University Thesis Abstract Page}

This public project demonstrates a reusable thesis workflow for the Master of
Engineering Management scenario at Nanjing University. The layout baseline is
derived from the public Word attachment shared in issue \#37, while the actual
editable scaffold is rebuilt under the `bensz-thesis` package model. The sample
intentionally separates instructional callouts from the real editable body so
that future users can keep the verified visual reference and still start from a
clean LaTeX source tree.
