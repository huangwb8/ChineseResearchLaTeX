# NSFC 地区科学基金项目申请书 - AI 写作指令

本目录是 **国家自然科学基金地区科学基金项目**申请书的 LaTeX 工作区。AI 在此的核心任务是**辅助撰写和优化标书内容**，不做版式开发，不改动 `extraTex/@config.tex`（除非用户明确要求调整版式）。

第（一）部分建议 **8000 字以下**；全文原则上不超过 **30 页**，鼓励简洁表达。

---

## 文件职责映射

所有正文内容集中在 `extraTex/`，与 `main.tex` 中的章节一一对应：

| 文件 | 对应章节 | 内容要点 |
|------|----------|----------|
| `extraTex/1.1.立项依据.tex` | （一）1. 项目的立项依据 | 研究意义 / 国内外现状 / 发展趋势 / 应用前景 |
| `extraTex/1.2.内容目标问题.tex` | （一）2. 研究内容、目标、关键问题 | 研究内容 / 研究目标 / 拟解决的关键科学问题 |
| `extraTex/1.3.方案及可行性.tex` | （一）3. 研究方案及可行性分析 | 研究方法 / 技术路线 / 实验手段 / 关键技术 |
| `extraTex/1.4.特色与创新.tex` | （一）4. 本项目的特色与创新之处 | 1–2 条高含金量创新点 |
| `extraTex/1.5.研究计划.tex` | （一）5. 年度研究计划及预期结果 | 三年分年度计划 / 预期成果 / 学术交流计划 |

## 写作顺序建议

```
明确科学问题
  → 立项依据（1.1）
  → 研究内容 + 目标 + 关键问题（1.2）
  → 研究方案及可行性（1.3）
  → 特色与创新（1.4）
  → 年度研究计划（1.5）
  → 研究基础（2.1） + 工作条件（2.2）
  → 承担/完成项目（2.3, 2.4）
  → 其他说明（3.x）
  → 全文质量控制
```

> **注意**：地区科学基金与面上项目最大的结构差异在于，立项依据、研究内容、方案和计划**合并在第（一）部分**，统一控制在 8000 字以内。

## 各章节写作要点

### （一）立项依据 → `extraTex/1.1.立项依据.tex`

写作逻辑：**研究意义** → **国内外研究现状及发展动态分析** → **科学意义或应用前景**

- 参考文献随立项依据之后渲染，通过 `\cite{key}` 引用，文献条目维护在 `references/myexample.bib`
- 可结合"科学研究发展趋势"论述科学意义，或结合"国民经济和社会发展的关键需求"论述应用前景

### （一）研究内容、目标、关键问题 → `extraTex/1.2.内容目标问题.tex`

- 研究内容：此部分为**重点阐述内容**
- 研究目标：明确、可验证
- 关键科学问题：1–3 个，需与研究内容高度对应

### （一）研究方案及可行性 → `extraTex/1.3.方案及可行性.tex`

- 包含研究方法、技术路线、实验手段、关键技术
- 技术路线图可通过 `nsfc-roadmap` 技能生成

### （一）特色与创新 → `extraTex/1.4.特色与创新.tex`

- 聚焦科学意义而非技术手段，1–2 条，言简意赅

### （一）年度研究计划 → `extraTex/1.5.研究计划.tex`

- 三年分年度，预期产出（论文/方法/数据库等）具体可查
- 可包含重要学术交流活动、国际合作计划

### （二）研究基础 → `extraTex/2.1–2.4.*.tex`

- **2.1**：用证据链（代表性论文/项目/数据）证明申请人有能力完成本课题
- **2.2**：已具备条件 + 缺少的条件 + 解决方案，简洁务实
- **2.3–2.4**：如实填写，格式参照模板中的 `\NSFCBlankPara` 占位区

## 常用 LaTeX 命令速查

```latex
% 空白占位符（草稿期使用，填写内容后删除）
\NSFCBlankPara

% 正文文件开头推荐加（两端对齐 + 2em 段首缩进）
\justifying
\NSFCBodyText

% 列表项标题（加粗蓝色标题 + 正文内容）
\item \itemtitlefont{标题：}内容...

% 三级标题（自动编号 1.1 风格）
\subsubsection{标题}

% 四级标题（自动编号（1）风格）
\subsubsubsection{标题}

% 圈号行内小标题
\ssssubtitle{1}

% 插入图片
\begin{figure}[!th]
    \begin{center}
        \includegraphics[width=0.8\linewidth]{figures/图片名.jpg}
        \caption{图片说明文字}
        \label{fig:label}
    \end{center}
\end{figure}

% 引用文献
已有研究表明...\cite{bibkey}
```

## 编译说明

**完整仓库模式**（项目在 `ChineseResearchLaTeX/projects/NSFC_Local/`）：

```bash
python scripts/nsfc_build.py build --project-dir .
```

**手工兜底**（任意环境均可，需已安装 TeXLive + 中文字体）：

```bash
xelatex main.tex
bibtex main
xelatex main.tex
xelatex main.tex
```

编译输出：中间文件进入 `.latex-cache/`，最终 PDF 保留在项目根目录 `main.pdf`。

## 工程原则

- **只改内容文件**：仅修改 `extraTex/*.tex`、`references/myexample.bib`、`figures/`；不改 `@config.tex` 版式（除非用户明确要求）
- **先读后写**：修改前先读取目标文件，了解现有内容和占位符再动笔
- **不造假引用**：添加参考文献必须经 `nsfc-bib-manager` 核实，不虚构 BibTeX 条目
- **篇幅意识**：第（一）部分建议 8000 字以内；全文不超过 30 页；超出时优先用 `nsfc-length-aligner` 压缩
- **语言**：始终用简体中文与用户交流，撰写中文标书内容
